Política de Privacidade

Este documento, denominado Política de Privacidade, possui a finalidade de estabelecer as regras sobre o uso, armazenamento e tratamento dos dados e informações fornecidos pelos Usuários em site de propriedade da Culinária Premium Utilidades Domésticas Ltda. ("Culinária Premium").

Dessa forma, a presente Política se aplica aos Usuários que disponibilizam seus dados, na medida da necessidade para usufruir dos produtos comercializados pela Culinária Premium.

1. Privacidade e Segurança

A Culinária Premium preza pela privacidade, pela segurança de dados e pela transparência com os seus consumidores. Diante disso, a empresa elaborou este documento para divulgar e explicar como os dados pessoais dos Usuários serão tratados e quais são as medidas aplicadas para mantê-los seguros.

De acordo com a Lei Geral de Proteção de Dados - LGPD (Lei Federal nº 13.709 de 2018), a Inovashop será, na maior parte do tempo, a controladora das informações dos Usuários, sendo responsável por definir o que acontece com esses dados e por protegê-los.

3. Objetivo da Política

Nesse sentido, a Culinária Premium reafirma seu compromisso em zelar pelas informações coletadas. Este é o objetivo desta Política de Privacidade de Dados.

4. Definições Importantes

Para fins desta Política de Privacidade, aplicam-se as seguintes definições:

  • Cookies: arquivos enviados pelo servidor do site para o computador do Usuário, com a finalidade de identificar o computador e obter dados de acesso.
  • IP: conjunto de números que identifica o computador do Usuário na Internet.
  • Logs: registros de atividades do Usuário efetuadas no site.
  • Session ID: identificação da sessão do Usuário no site.
  • Usuário: todo aquele que passar a usar o site.
  • Tratamento de Dados Pessoais: operações realizadas com dados pessoais, como coleta, armazenamento, processamento, comunicação, entre outras.

5. Coleta de Dados

A Culinária Premium poderá coletar diferentes tipos de dados pessoais, de forma automática ou fornecidos diretamente pelo Usuário:

  • Nome completo, CPF, e-mail, número de celular, data de nascimento e endereço.
  • Dados do cartão de crédito e informações de pagamento.
  • Endereço de IP, produtos pesquisados, páginas visitadas.
  • Informações de login social, quando autorizado.
  • Documentos enviados pelo Usuário, como comprovantes e identidade.

6. Cookies

O site utiliza cookies para identificar o comportamento do Usuário e recomendar produtos de acordo com o seu perfil.

7. Plataforma de Pagamento

A Culinária Premium utiliza uma plataforma de pagamento operada por terceiros, que coleta informações relacionadas a dados de pagamento, como bandeira e número do cartão de crédito.

8. Direitos do Usuário

O Usuário pode adicionar, excluir ou modificar quaisquer informações relacionadas ao seu perfil junto à Culinária Premium. A transparência sobre o tratamento de dados pessoais é prioridade.

9. Finalidade do Tratamento de Dados

A Culinária Premium utiliza os dados pessoais dos Usuários para:

  • Identificar corretamente o Usuário.
  • Efetuar comunicações relacionadas ao site.
  • Enviar produtos adquiridos e informações sobre ofertas.
  • Desenvolver novas funcionalidades e melhorias no site.
  • Atender a ordens judiciais ou regulamentações.

10. Armazenamento e Segurança

A Culinária Premium adota as melhores práticas para proteger os dados pessoais coletados. No entanto, nenhuma plataforma é completamente segura. Caso o Usuário tenha preocupações, deve entrar em contato.

11. Compartilhamento de Dados

Os dados pessoais podem ser compartilhados com prestadores de serviços, autoridades judiciais ou reguladoras, sempre que necessário, mas nunca serão comercializados.

12. Alterações na Política

Esta Política de Privacidade poderá ser atualizada ou modificada a qualquer momento. O Usuário deverá consultá-la regularmente.
